## Что выведет консоль?

const obj = { a: 1, b: 2, c: 3 };
const { a, ...rest } = obj;
console.log(a, rest);

Ответ:

1 { b: 2, c: 3 }

Разбираем объект с помощью деструктуризации: a = 1, rest = { b: 2, c: 3 }.

## 2. Что выведет консоль?

const multiply = (a, b = 2) => a \* b;
console.log(multiply(5));
console.log(multiply(5, 3));

Ответ:

10
15

Если b не передано, используется значение по умолчанию 2.

## 3. Что будет в переменной `y`?

let x = [1, 2, 3];
let y = x.map(n => n \* 2);

Ответ:

[2, 4, 6]

Метод .map() создаёт новый массив, умножая каждый элемент на 2.

## 4. Что выведет консоль?

const sum = (...args) => args.reduce((a, b) => a + b, 0);
console.log(sum(1, 2, 3, 4));

Ответ:

10

Используем rest-параметр и .reduce() для суммирования.

## 5. Какое значение у `z`?

let z = 10;
z = z || 20;

Ответ:

10

Оператор || возвращает первое "truthy" значение, то есть 10.

## 6. Что выведет консоль?

let x = 'var';
(function() {
console.log(x);
let x = 'let';
})();

Ответ:

Uncaught ReferenceError: Cannot access 'x' before initialization

Переменная x поднимается (hoisting), но не инициализируется до console.log(x).

## 7. Что выведет консоль?

let a = [1, 2, 3];
let b = a;
b.push(4);
console.log(a);

Ответ:

[1, 2, 3, 4]

b и a ссылаются на один и тот же массив, поэтому изменение b затрагивает a.

## 8. Что выведет консоль?

const user = {
name: 'Alice',
greet: function() {
console.log('Hello, ' + this.name);
}
};

const greet = user.greet;
greet();

Ответ:

Hello, undefined

Функция greet вызывается без контекста (`this` = undefined в строгом режиме).

## 9. Что выведет консоль?

let arr = [10, 20, 30];
arr.length = 2;
console.log(arr);

Ответ:

[10, 20]

Установка arr.length = 2 обрезает массив.

## 10. Что выведет консоль?

console.log(typeof null);
console.log(typeof undefined);
console.log(null === undefined);

Ответ:

"object"
"undefined"
false

typeof null — баг JavaScript, null и undefined — разные типы.

## 11. Что выведет консоль?

const a = [1, 2, 3];
const b = [1, 2, 3];

console.log(a == b);
console.log(a === b);

Ответ:

false
false

Массивы сравниваются по ссылке, а a и b — разные объекты в памяти.

## 12. Что выведет консоль?

const foo = {a: 1};
foo.b = 2;
console.log(foo);

Ответ:

{ a: 1, b: 2 }

const защищает только саму переменную, но не её свойства.

## 13. Разница между `map()` и `forEach()`?

array.map() // Возвращает новый массив
array.forEach() // Ничего не возвращает

Ответ:  
`forEach()` ничего не возвращает, а `map()` возвращает новый массив.

## 14. Что выведет консоль?

console.log([..."12345"]);

Ответ:

['1', '2', '3', '4', '5']

Оператор ... разбивает строку на массив символов.
